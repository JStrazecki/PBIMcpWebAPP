# MCP Server Startup Commands

## Option 1: Authenticated MCP Server (Microsoft OAuth Required)
# Users must login with Microsoft credentials to access
gunicorn --bind=0.0.0.0:8000 --workers 1 --timeout 600 --access-logfile - --error-logfile - --log-level info mcp_auth_server:app

# Environment variables required:
# AZURE_CLIENT_ID=your-client-id
# AZURE_CLIENT_SECRET=your-client-secret  
# AZURE_TENANT_ID=your-tenant-id

## Option 2: Simple MCP Server (No Authentication)
# Direct connection, no login required
gunicorn --bind=0.0.0.0:8001 --workers 1 --timeout 600 --access-logfile - --error-logfile - --log-level info mcp_simple_server:app

# No environment variables required

## Azure Deployment
# For Azure Web Apps, replace port with :

# Authenticated server:
gunicorn --bind=0.0.0.0: --workers 2 --timeout 600 --access-logfile - --error-logfile - --log-level info mcp_auth_server:app

# Simple server:
gunicorn --bind=0.0.0.0: --workers 2 --timeout 600 --access-logfile - --error-logfile - --log-level info mcp_simple_server:app
